# Web

This skill provides access to external technical information and research.

## Available Tools

- `web_search` — Search the web for documentation, APIs, troubleshooting resources, and implementation references.

## Instructions

Use `web_search` to:

- Find official documentation for APIs, SDKs, frameworks, or libraries
- Research unfamiliar technologies or integrations
- Investigate complex errors that require external context
- Validate implementation approaches against authoritative sources
- Find troubleshooting guides, examples, or reference material

Prefer this skill when the required answer depends on external knowledge rather than only the local codebase.
