This skill provides capabilities for reading and modifying files in the sandbox.

## Tools
- `read_file`: Read the full contents of a file at an absolute path.
- `edit_file`: Create or modify a file (overwrite, append, or replace).

## Usage
Use this skill whenever you need to explore the codebase, initialize projects, or write code.
