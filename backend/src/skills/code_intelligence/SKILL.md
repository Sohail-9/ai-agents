# Code Intelligence

This skill helps inspect and understand the current codebase.

## Available Tools

- `search_code` — Search text patterns or regular expressions across repository files.

## Instructions

Use `search_code` to:

- Find function, class, or variable definitions
- Locate references or usages across the codebase
- Trace implementation flow between modules
- Inspect configuration or dependency usage
- Identify impacted files for debugging or refactoring

Prefer this skill whenever the task requires repository-wide code discovery rather than reasoning from provided snippets alone.
